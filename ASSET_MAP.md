#### Pretext 阅读器方案

| 维度 | 说明 |
|------|------|
| **技术栈** | `@chenglou/pretext` (MIT, 47.6k⭐) + TypeScript |
| **核心能力** | 零DOM文字测量、Knuth-Plass段落排版、富文本内联、Markdown虚拟化 |
| **为什么用它** | 公众号排版受限（单栏、无高级排版），Pretext可以做一个媲美电子书的独立阅读器 |
| **渲染效果** | 紧凑多行、防"河流"对齐、代码+文字混排、章节间虚拟滚动 |
| **部署方式** | 静态HTML页面 → 可部署到 GitHub Pages / EdgeOne Pages |
| **依赖** | 浏览器需支持 `Intl.Segmenter` + Canvas 2D（现代浏览器均支持） |
